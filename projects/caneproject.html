The Idea

This website uses the Twitter REST API to import youtube videos relating to top sports plays. I worked on this project at Bitcamp (University of Maryland's first hackathon event) with two friends. We developed the back-end of the site using PHP, with HTML and CSS to structure and style the site. At first we considered using Ruby on Rails to develop the site, but we couldn't find enough resources for the Twtter Ruby Gems we wanted to use.
